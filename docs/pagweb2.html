¿QUE ES HTML5?

HTML 5

es la quinta revisión importante del lenguaje básico de la World Wide Web, HTML. HTML5 específica dos variantes de sintaxis para HTML: una «clásica», HTML (text/html), conocida como HTML5, y una variante XHTML conocida como sintaxis XHTML 5 que deberá servirse con sintaxis XML (application/xhtml+xml). Esta es la primera vez que HTML y XHTML se han desarrollado en paralelo. La versión definitiva de la quinta revisión del estándar se publicó en octubre de 2014. Al no ser reconocido en viejas versiones de navegadores por sus nuevas etiquetas, se recomienda al usuario común actualizar su navegador a la versión más nueva, para poder disfrutar de todo el potencial que provee HTML 5. El desarrollo de este lenguaje de marcado es regulado por el Consorcio W3C.

HTML5 establece una serie de nuevos elementos y atributos que reflejan el uso típico de los sitios web modernos. Algunos de ellos son técnicamente similares a las etiquetas "div" y "span", pero tienen un significado semántico, como por ejemplo "nav" (bloque de navegación del sitio web) y "footer".

Novedades

  1. Incorpora etiquetas (canvas 2D y 3D, audio, vídeo) con codecs para mostrar los contenidos multimedia. Actualmente hay una lucha entre imponer codecs libres (WebM + VP8) o privados
  2. Etiquetas para manejar grandes conjuntos de datos: Datagrid, Details, Menu y Command. Permiten generar tablas dinámicas que pueden filtrar, ordenar y ocultar contenido en cliente.
  3. Mejoras en los formularios. Nuevos tipos de datos (eMail, number, url, datetime …) y facilidades para validar el contenido sin Javascript.
  4. Visores: MathML (fórmulas matemáticas) y SVG (gráficos vectoriales). En general se deja abierto a poder interpretar otros lenguajes XML.
  5. Drag & Drop. Nueva funcionalidad para arrastrar objetos como imágenes.

¿Cuales son sus ventajas?

ofrece a los programadores de sitios web nuevas oportunidades. Así, navegadores web tan populares como Internet Explorer, Mozilla Firefox o Google Chrome ya son compatibles con HTML 5. Además, este estándar también funciona correctamente con smartphones y tablets. De este modo, también es posible mejorar la velocidad y visualización de las páginas webs en dispositivos móviles. Con HTML 5, cualquier programador puede integrar las denominadas “Rich Internet Application” (RIA) e incluir vídeos directamente en el código fuente de la página. El navegador puede, así, reproducir vídeos, audio e incluso juegos. Para dispositivos móviles esto representa una gran ventaja. Hasta HTML 5, Flash ha sido dueño y señor de este sector, pero le queda poco tiempo.

¿Cómo funciona y para qué sirve el lenguaje HTML5?

Lo primero que se necesita es un navegador web y un servidor. El navegador envía una petición al servidor (con el formato https://…) y el servidor busca esa información en sus archivos para mostrar la página requerida al navegador. Además, HTML5 no solo es la última versión de HTML, sino que incluye un grupo de especificaciones que permiten un mejor desarrollo web y formato en su percepción final, como pueden ser XHTML1, CSS y DOM. Por lo tanto, no solo podrás crear la estructura de una web, sino que tendrás acceso a otras opciones como la inclusión de audio y vídeo sin necesidad de usar Flash. Tomando este último ejemplo, y usando las etiquetas antes mencionadas, al insertar "video" y "audio" podremos reproducir vídeos y audios directamente desde el navegador, por lo que el peso de la web y su carga serán menores. También puedes insertar, a través de otras etiquetas, archivos multimedias alojado en otras plataformas como YouTube. Pero esto no es todo, el lenguaje HTML5 también permite detectar la ubicación del visitante que accede a la página. Esto puede resultar muy útil para, por ejemplo, mostrar la web en un idioma concreto o mejorar la experiencia del usuario mostrándole contenido local. Por supuesto, debe hacerse informando siempre al usuario y dentro de las legalidades vigentes.

¿Qué significan las siglas HTML?

Las siglas HTML provienen del inglés: HyperText Markup Language, y esta se trata de la quinta versión de este lenguaje que se utiliza en todo el mundo para crear la infraestructura de prácticamente cualquier página web. Gracias a esto se pueden ver textos, imágenes, vídeos y mucho más en las pantallas.

  • HyperText: Un hipertexto es, básicamente, un texto que puede enlazarse con otros contenidos. Esta es la base del internet que conocemos hoy en día, con una red de páginas, archivos y otros recursos conectados unos a otros.
  • Markup: Las etiquetas son parte de la cara interna de una página web, que indica al navegador cómo mostrar algo concreto. Por ejemplo, la etiqueta "i" y "/i" marcan un texto que se verá en cursiva.
  • Language: Se denomina lenguaje porque, como cualquier otro en el mundo, tiene sus normas, su estructura a la hora de escribirlo y una serie de cuestiones a tener en cuenta. Así, todo el mundo que conozca el lenguaje HTML podrá entender una página web

Se debe tener en cuenta que definir el HTML como un lenguaje no lo convierte en un lenguaje de programación (como sí lo son JavaScript, Python, C + +, Visual Basic…). Estos últimos tienen una serie de condicionantes, bucles y otras funciones que no aparecen en el lenguaje de etiquetas.
